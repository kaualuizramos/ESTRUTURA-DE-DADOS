# 📚 Estrutura de Dados

Repositório dedicado aos estudos da disciplina de **Estrutura de Dados**, contendo implementações e exercícios desenvolvidos durante as aulas.

## 📌 Conteúdo

Neste repositório você encontrará implementações de estruturas de dados fundamentais, como:

- Listas Ligadas
- Listas Duplamente Ligadas
- Pilhas
- Filas
- Árvores
- Algoritmos de busca
- Algoritmos de ordenação

Os códigos são implementados principalmente em **C**, com foco em compreender conceitos como:

- Ponteiros
- Alocação dinâmica de memória
- Manipulação de estruturas
- Implementação de TADs (Tipos Abstratos de Dados)

## 🚀 Tecnologias

![C](https://img.shields.io/badge/C-grey)

## 🎯 Objetivo

O objetivo deste repositório é:

- Praticar conceitos fundamentais de **Estrutura de Dados**
- Melhorar a lógica de programação
- Aprender a implementar estruturas manualmente
- Servir como material de consulta para estudos futuros
- Demonstrar Habilidades com a linguagem C

## 📂 Organização

Os arquivos estão organizados por desafios de estrutura de dados, facilitando a navegação e o estudo.
